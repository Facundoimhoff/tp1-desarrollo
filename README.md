TP1 - Desarrollo Web

1- Descripción: 
Este proyecto corresponde al **Trabajo Práctico Nº1** de la materia **Desarrollo Web**.
Consiste en la creación de un **portfolio personal y profesional**, desarrollado utilizando tecnologías web básicas.

2- Tecnologías utilizadas:

* HTML5
* CSS3

3- Estructura del proyecto:

* `index.html` → Página principal del portfolio
* `CSS/` → Archivos de estilos
* `IMAGENES/` → Recursos gráficos utilizados en el sitio

4- Objetivo:

Aplicar los conocimientos fundamentales de desarrollo web, incluyendo:

* Estructura semántica en HTML
* Estilos y diseño visual con CSS
* Organización de archivos en un proyecto web
